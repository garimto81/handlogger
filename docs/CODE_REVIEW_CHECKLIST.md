## 1. 코드 크기 검증

### 1.1 라인 수 측정
```bash
# 현재 라인 수 확인
wc -l code.gs

# 변경된 라인 수 확인
git diff HEAD -- code.gs | grep -E "^\+[^+]" | wc -l  # 추가
git diff HEAD -- code.gs | grep -E "^-[^-]" | wc -l  # 삭제
```

### 1.2 함수 개수 측정
```bash
grep "^function" code.gs | wc -l
```
